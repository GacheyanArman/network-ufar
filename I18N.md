# Система интернационализации (i18n)

Документация по многоязычной поддержке приложения.

## Поддерживаемые языки

- 🇬🇧 **English (EN)** - Английский
- 🇫🇷 **Français (FR)** - Французский
- 🇦🇲 **Հայերեն (HY)** - Армянский

## Архитектура

### 1. Файл переводов (`src/lib/i18n.ts`)

Содержит все переводы для всех языков в структурированном формате:

```typescript
export const translations = {
  en: {
    nav: {
      myNews: "My News",
      search: "Search",
      // ...
    },
    common: {
      logout: "Logout",
      save: "Save",
      // ...
    },
  },
  fr: { /* ... */ },
  hy: { /* ... */ },
};
```

### 2. Language Context (`src/contexts/LanguageContext.tsx`)

React Context для управления текущим языком:

```typescript
const { language, setLanguage, t } = useLanguage();
```

**Функции:**
- `language` - текущий выбранный язык
- `setLanguage(lang)` - изменить язык
- `t(key)` - получить перевод по ключу

### 3. Language Switcher (`src/components/LanguageSwitcher.tsx`)

Компонент переключателя языков с выпадающим списком.

### 4. Navigation Menu (`src/components/NavigationMenu.tsx`)

Пример использования переводов в компоненте навигации.

## Использование

### В клиентских компонентах

```typescript
"use client";

import { useLanguage } from "@/contexts/LanguageContext";

export default function MyComponent() {
  const { t } = useLanguage();

  return (
    <div>
      <h1>{t("nav.myNews")}</h1>
      <button>{t("common.save")}</button>
    </div>
  );
}
```

### Структура ключей переводов

Используйте точечную нотацию для доступа к вложенным переводам:

```typescript
t("nav.myNews")           // "My News" / "Mes Actualités" / "Իմ Նորությունները"
t("common.logout")        // "Logout" / "Déconnexion" / "Դուրս Գալ"
t("feed.createPost")      // "Create Post" / "Créer une Publication" / "Ստեղծել Գրառում"
t("events.title")         // "University Events" / "Événements Universitaires" / "Համալսարանական Իրադարձություններ"
```

## Добавление новых переводов

### Шаг 1: Добавьте ключи в `src/lib/i18n.ts`

```typescript
export const translations = {
  en: {
    mySection: {
      myKey: "My English Text",
    },
  },
  fr: {
    mySection: {
      myKey: "Mon Texte Français",
    },
  },
  hy: {
    mySection: {
      myKey: "Իմ Հայերեն Տեքստը",
    },
  },
};
```

### Шаг 2: Используйте в компоненте

```typescript
const { t } = useLanguage();
<p>{t("mySection.myKey")}</p>
```

## Сохранение выбранного языка

Язык автоматически сохраняется в `localStorage` и восстанавливается при следующем посещении:

```typescript
// Автоматически в LanguageContext
localStorage.setItem("language", lang);
```

## Fallback на английский

Если перевод не найден для выбранного языка, система автоматически использует английский вариант:

```typescript
// Если hy.someKey не существует, вернется en.someKey
t("someKey") // fallback to English
```

## Категории переводов

### `nav` - Навигация
Все пункты меню навигации

### `common` - Общие элементы
Кнопки, действия, общие фразы

### `feed` - Лента новостей
Тексты для главной ленты

### `profile` - Профиль
Тексты для страницы профиля

### `events` - События
Тексты для системы событий

### `calendar` - Календарь
Тексты для академического календаря

### `widgets` - Виджеты
Тексты для боковых виджетов

## Примеры использования

### Пример 1: Кнопка с переводом

```typescript
"use client";

import { useLanguage } from "@/contexts/LanguageContext";

export default function SaveButton() {
  const { t } = useLanguage();

  return (
    <button className="btn btn-primary">
      {t("common.save")}
    </button>
  );
}
```

### Пример 2: Заголовок страницы

```typescript
"use client";

import { useLanguage } from "@/contexts/LanguageContext";

export default function EventsPage() {
  const { t } = useLanguage();

  return (
    <div>
      <h1>{t("events.title")}</h1>
      <p>{t("events.description")}</p>
    </div>
  );

```

### Пример 3: Динамический список

```typescript
"use client";

import { useLanguage } from "@/contexts/LanguageContext";

export default function EventTypeSelector() {
  const { t } = useLanguage();

  const eventTypes = ["party", "academic", "sports", "cultural", "workshop", "other"];

  return (
    <select>
      {eventTypes.map((type) => (
        <option key={type} value={type}>
          {t(`events.eventTypes.${type}`)}
        </option>
      ))}
    </select>
  );
}
```

## Рекомендации

### ✅ Хорошие практики

1. **Используйте осмысленные ключи**
   ```typescript
   t("profile.editProfile") // ✅ Понятно
   t("btn1") // ❌ Непонятно
   ```

2. **Группируйте связанные переводы**
   ```typescript
   events: {
     title: "...",
     description: "...",
     eventTypes: { ... }
   }
   ```

3. **Добавляйте переводы для всех языков сразу**
   Не оставляйте пустые переводы

4. **Используйте fallback**
   Система автоматически вернется к английскому

### ❌ Избегайте

1. Не хардкодите тексты в компонентах
2. Не дублируйте одинаковые переводы
3. Не используйте слишком длинные ключи
