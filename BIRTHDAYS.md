# Birthday Feature Documentation

## Обзор

Функционал дней рождения позволяет пользователям указывать свою дату рождения в профиле и видеть, у кого из друзей сегодня день рождения.

## Реализованные возможности

✅ **Поле birthDate в базе данных**
- Добавлено поле `birth_date` типа `timestamp` в таблицу `user`
- Миграция: `drizzle/0005_eager_network.sql`

✅ **Редактирование даты рождения**
- Поле выбора даты в форме редактирования профиля
- Тип input: `date` с ограничением максимальной даты (сегодня)
- Сохранение через server action `updateProfile`

✅ **Виджет Birthdays в правом сайдбаре**
- Показывает друзей, у которых сегодня день рождения
- Максимум 5 пользователей
- Сравнение только по дню и месяцу (год не учитывается)
- Ссылки на профили пользователей
- Аватары и имена

✅ **Логика определения дней рождения**
- Функция `getTodayBirthdays()` в `src/lib/birthdays.js`
- Проверяет только друзей текущего пользователя (статус friendship = "accepted")
- Использует SQL EXTRACT для сравнения месяца и дня
- Не показывает пользователей без указанной даты рождения

## Использование

### Установка даты рождения

1. Перейти в профиль → Edit profile
2. Заполнить поле "Birth Date"
3. Сохранить изменения

### Просмотр дней рождения

Виджет "Birthdays" в правом сайдбаре показывает:
- **Если есть дни рождения сегодня**: список друзей с аватарами и надписью "🎂 Happy Birthday!"
- **Если нет дней рождения**: "No birthdays today."

### SQL запрос

```sql
-- Получить пользователей с днем рождения сегодня
SELECT id, fullName, image, birth_date
FROM "user"
WHERE 
  id IN (список друзей)
  AND birth_date IS NOT NULL
  AND EXTRACT(MONTH FROM birth_date) = текущий_месяц
  AND EXTRACT(DAY FROM birth_date) = текущий_день
LIMIT 5;
```

## Дизайн

Виджет выполнен в стиле UFARnet:
- Французская элегантная палитра
- Закругленные углы (border-radius: 12px)
- Hover-эффекты с плавной анимацией
- Аватары круглые (50%)
- Иконки через UiIcon компонент

## Примечания

- Год рождения не влияет на отображение — сравниваются только день и месяц
- Показываются только друзья (не подписчики)
- Пользователи без указанной даты рождения не отображаются
- Максимум 5 пользователей в виджете
- Дата рождения видна всем пользователям (публичное поле)
