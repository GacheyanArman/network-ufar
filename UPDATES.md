## 📊 Структура БД

Таблица `comment` уже существует в схеме:
```sql
CREATE TABLE comment (
  id TEXT PRIMARY KEY,
  post_id TEXT NOT NULL REFERENCES post(id) ON DELETE CASCADE,
  author_id TEXT NOT NULL REFERENCES user(id) ON DELETE CASCADE,
  content TEXT NOT NULL,
  created_at TIMESTAMP DEFAULT NOW() NOT NULL
);
```

## 🚀 Как использовать

### Комментарии
1. Нажмите на кнопку "Comments" под постом
2. Введите текст комментария (макс. 1000 символов)
3. Нажмите кнопку отправки (иконка самолетика)
4. Комментарий появится мгновенно (оптимистичное обновление)

### Дизайн
- Все цвета автоматически обновлены
- Градиенты применяются к кнопкам и активным элементам
- Золотые акценты на активной навигации
